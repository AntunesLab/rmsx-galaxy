# Changelog

## 0.1.0+galaxy0

- Pin the RMSX runtime scaffold to upstream RMSX `v0.2.3`.
- Document that the `v0.2.3` source still installs package metadata as
  `rmsx==0.1.0`.

- Keep the custom `rmsxmolstar` datatype and native Molstar visualization as
  prototype/cofest assets pending Galaxy/IUC guidance.
- Add explicit Tool Shed readiness documentation, test-data provenance notes,
  and IUC review concerns.
